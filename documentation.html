Mediotec Login System Documentation

React + Cypress based login system with full technical documentation

• README

Welcome to the Mediotec Login System! This project is a simple and effective login page developed using React and tested with Cypress.

  • Responsive login page using React
  • Cypress end-to-end testing
  • Custom styles with CSS
  • Functional input fields and form handling

This project was created for a Tech English documentation assignment.

• Installation Guide

  1. Clone the repository: 
    git clone https://github.com/your-user/seu-projeto.git
cd seu-projeto
  2. Install dependencies: 
    npm install
  3. Start the development server: 
    npm run dev

    Access http://localhost:5173 in your browser.

Running Tests

  • Install Cypress: 
    npm install --save-dev cypress
  • Open Cypress: 
    npx cypress open
  • Run tests: 
    npm run test
    npm run test:headed
    npm run test:open

• API Reference

This is a front-end-only project. There are no API endpoints. The behavior is defined by React components and Cypress test cases.

Form Elements

ElementDescription

Test File

Located at cypress/e2e/login.cy.js. It tests visibility, typing, and form submission with success message.
